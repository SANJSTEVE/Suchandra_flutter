# Product

An interactive single-page web portfolio that converts a traditional resume into a dynamic, visually compelling personal brand hub. It presents professional experience, skills, projects, and education with scroll-triggered animations, dark/light theming, and project filtering — deployable as a static site with no backend dependencies.

## Key Capabilities
- Full resume content rendered from a JSON data source
- Dark/light theme toggle with localStorage persistence
- Tag-based project filtering with animated transitions
- Scroll-triggered section entrance animations
- Responsive layout for mobile, tablet, and desktop
- Optional contact form with client-side validation
- Zero backend required — deployable to GitHub Pages, Vercel, or Netlify
